## 常用标签

### 标题

```html
    <h1>This is a title</h1>
    <h2>This is a title</h2>
    <h3>This is a title</h3>
    <h4>This is a title</h4>
    <h5>This is a title</h5>
    <h6>This is a title</h6>
```

**演示:**

    <h1>This is a title</h1>
    <h2>This is a title</h2>
    <h3>This is a title</h3>
    <h4>This is a title</h4>
    <h5>This is a title</h5>
    <h6>This is a title</h6>
    
### 空行

> html 所有标签都需进行闭合, 请在使用 `<br>` 标签时同样将其闭合为 `<br />`

```html
    <br />
```

**演示:**
    
    <br />
